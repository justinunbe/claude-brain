# Weekly Planning Ritual

## Cadence

TBD — Jay to set day/time on first run. Reference: Jay's old 2021 Weekly Review split Career + Life and ran Sundays. Jay's MAR 8 note framed planning as Reactive (organize/distill/manage/share information) and Proactive (push accounts, internal projects, manage up).

## Shape (v1 — will evolve after first run)

### Inputs (Claude pulls before the session starts)

- `state/current-hare.md` — what's the active focus
- `state/active-goals.md` — where we are on specifics
- Last 1-2 `sessions/*` entries — recent context
- `profile/from-jay.md` — anything Jay dropped in the backchannel
- Last week's weekly plan (if one exists) — what did we commit to, what actually happened
- Calendar ahead (if connected later) — what's structural next week

### Agenda

1. **Look back at last week** — What moved forward? What slipped? What did we learn?
2. **Check the hare** — Still Health? Progress toward 12% BF? Any shifts for `state/shifts.md`?
3. **Quick pass across the 5 Areas** — Any tortoise needing a bump? Anything falling behind?
4. **Open loops** — What's hanging? What needs a decision?
5. **Commit to next week** — Top 3-5 things that matter most. Specific, measurable where possible.
6. **Surface anything Jay wants captured** — feelings, observations, ideas.

### Outputs

- New file: `sessions/YYYY-MM-DD-weekly-plan-W##.md` — the plan itself.
- Updates to `state/active-goals.md` — new goals added, completed items checked off.
- New entries in `state/shifts.md` if Jay's framing has shifted.
- `profile/from-jay.md` entries read and marked `[DONE YYYY-MM-DD]`.

## Discipline

- Keep it under 30 minutes unless Jay explicitly wants depth.
- Ask only questions that can't be inferred from files — Jay should never re-explain what's already documented.
- Surface contradictions without judgment ("last week you said X, this week you're saying Y — which is it?").
- End with the end-of-session synthesis pattern (see `rituals/session-workflow.md`).

## Open questions (answered during first run)

- Day / time / cadence.
- Format of the plan file — narrative vs. structured? Default: structured.
- Does Jay want to see the plan before it's committed, or just talk it through?
